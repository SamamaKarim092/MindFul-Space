# 🧠 Mindful Space – Mental Health Sentiment Journal
**Mindful Space** is a full-stack journaling web application focused on emotional well-being. It allows users to reflect on their thoughts, track moods, and receive uplifting support — all in a safe, personalized space.

🔗 **Live Demo**:  
👉 https://mental-health-sentiment-journal.onrender.com/
---

## 📌 Key Features

- 📝 **User Journal Entries**  
  Users can write daily reflections and emotional thoughts in a personal journal.

- 📅 **Entry History & Date Filtering**  
  View, edit, or delete past journal entries. Filter entries by date to analyze your emotional journey.

- 🔐 **Secure Authentication**  
  - Uses **Firebase Authentication** for login/signup.
  - Keeps user data private and accessible only to the owner.

- ☁️ **Cloud Database (MongoDB Atlas)**  
  - Stores journal data on **MongoDB Atlas**, a cloud-hosted, scalable database.
  - Data routing handled via **Express.js** for secure transactions.

---

## 📊 Mood Charts & Emotional Trends

- Visualizes emotional patterns over time using **Chart.js**.
- Charts are:
  - 📈 Line-based or pie-based
  - 🎨 Color-coded: 🟢 Positive | 🟡 Neutral | 🔴 Negative

---

## 🌈 Daily Uplifting Quotes & Mental Health Tips

- Displays motivational content or tips upon login or when opening the journal.
- Boosts mindfulness and encourages positive habits.

---

## 🤖 Gemini AI Chat Assistant

- Integrated **Gemini AI** (by Google) to offer conversational support.
- The assistant can:
  - Provide gentle encouragement
  - Suggest self-care strategies
  - Simply offer a space to "talk"
